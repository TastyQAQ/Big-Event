## 安裝

```bash
# 取得專案
git clone https://github.com/TastyQAQ/Big-Event
```

```bash
# 安裝套件
pnpm install
```

```bash
# 運行專案
pnpm dev
```

## 主要工具

* Vue v3.3.4
* Vue Router v4.2.4
* Pinia v2.1.6
* element-plus v2.3.14
* axios v1.5.1
